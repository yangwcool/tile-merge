# tile-merge
拼接卫星图瓦片的工具，带一个Tk界面。

## 用法
1. 把下载到的瓦片放到一个文件夹里。要求：每一行（y坐标一样）瓦片放到一个文件夹里，文件夹名字为行号。瓦片文件名为列号（x坐标）。例如：
```
-tiles
  -1000
    -1.jpg
    -2.jpg
    -3.jpg
  -1001
    -1.jpg
    -2.jpg
    -3.jpg
  -1002
    -1.jpg
    -2.jpg
    -3.jpg
```

2. 运行脚本，填写瓦片y坐标范围和x坐标范围（最大和最小值）、瓦片尺寸、文件格式等。
3. 点【确认参数】，然后点【开始拼接】即可。
## 注意
本程序只能拼接瓦片，不能下载瓦片！
